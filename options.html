RegExp Filters

Whitelist (inclusion)

If at least one regex is defined here, only matching URLs will be kept.
Examples:
^https://example\.com
/articles/
\.pdf$
^https://(www\.)?github\.com

Blacklist (exclusion)

URLs matching these regexes will be excluded after applying the whitelist.
Examples:
\?utm_
#
/logout
\.png$
\.jpg$
If checked, when an exact URL is not found but a URL in its sub-path is found, the child URL's visit date will be used for the queried URL.
